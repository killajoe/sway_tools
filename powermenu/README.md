# powermenu - a very simple rofi powermenu

## needs rofi config:
`~/.config/rofi/powermenu.rasi`
or systemwide: `/usr/share/sway_tools/rofi/powermenu.rasi`

In case used with [garcefully_exit](https://github.com/killajoe/sway_tools/tree/main/gracefully_exit)
when taking sway-powermenu-gracefully script instead of normal sway-powermenu.
